# aitos.io Brand Assets — Wordmark System (V2.0, 2026)

Tagline: **Data stays sovereign. Value moves freely.**

## The mark
Zero graphic ornament — the type IS the brand.
Wordmark: lowercase "aitos.io", Unbounded Medium (500), letter-spacing +0.02em.
The only design move: ".io" in Signal Cyan (#5CE1FF on dark; #0891B8 on light for contrast).
Monogram: Unbounded lowercase "a" + a cyan dot in the lower-right quadrant (a distilled ".io").

## Files
- wordmark-dark.png — primary, for dark backgrounds (transparent, hi-res)
- wordmark-light.png — for light backgrounds (Ink + deep cyan)
- wordmark-mono.png — single-color version (stamps/engraving/fax)
- app-icon-512.png — rounded-square monogram (Deep Space)
- social-avatar-512.png — circular monogram
- favicon-32.png / favicon-180.png (Apple touch)

## Color
#05070D Deep Space · #5CE1FF Signal Cyan (.io, data/trust) · #0891B8 Cyan Deep (light bg) · #FFB454 Value Amber (product/web only — never in the logo) · #EDF2F8 Ink Light

## Rules
1. Never color the whole word / never use Amber in the logo.
2. Lowercase only; weight 500 (never 600+).
3. Never stretch. 4. No glow, shadow, or outline.
Fonts: Unbounded (logo/display), Instrument Sans (body/UI), IBM Plex Mono (labels/data) — all Google Fonts.
Minimum sizes: wordmark 14px cap height; favicon 16px.
