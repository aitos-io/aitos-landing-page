01  —  The Wordmark
aitos.io

Unbounded Medium 500 · lowercase · letter-spacing +0.02em · ".io" → Signal Cyan

The wordmark is the entire identity: lowercase "aitos.io" set in Unbounded Medium. The only design move is ".io" in Signal Cyan — the interface between the physical and digital worlds, rendered in the color of data.

The monogram distills the mark to a lowercase "a" with a cyan dot in the lower-right quadrant — a compressed ".io" for avatars, app icons and favicons.

05  —  Rules
✓

Lowercase only, weight 500. Color only the ".io" — Signal Cyan on dark, Cyan Deep on light.

✓

Minimum sizes: wordmark 14px cap height; favicon 16px.

✕

Never color the whole word. Never use Value Amber in the logo.

✕

Never stretch, no glow, no shadow, no outline, no weights above 500.
